# PEST 宏观环境分析框架（汽车行业版）

## 1. 适用场景

**何时使用此框架**：
- 分析宏观环境对汽车市场的影响
- 评估政策变化对车企战略的影响
- 识别市场机会与威胁
- 制定中长期战略规划

**问题类型**：
- "政策对新能源市场的影响"
- "经济增长对购车需求的影响"
- "技术趋势对行业格局的影响"

## 2. 分析维度

### P - 政治/政策 (Political)

| 分析维度 | 具体指标 | 数据来源 | 量化要求 |
|---------|---------|---------|---------|
| 购置税减免 | 新能源汽车购置税减免政策延续情况 | gov.cn | 具体政策内容和有效期 |
| 双积分 | 双积分政策走向、企业积分余量 | 工信部 | 企业积分计算 |
| 地方补贴 | 地方补贴退坡节奏、以旧换新政策 | gov.cn | 各城市补贴金额 |
| 碳排放法规 | 国六B、国七排放法规进度 | 官方公告 | 实施时间表 |
| 进口关税 | 进口关税调整、零部件关税 | 国务院公告 | 税率和影响范围 |
| 贸易政策 | 出口关税、跨境电商政策 | 商务部 | 具体政策 |

### E - 经济 (Economic)

| 分析维度 | 具体指标 | 数据来源 | 量化要求 |
|---------|---------|---------|---------|
| GDP增速 | GDP增长率、消费者信心指数 | 国家统计局 | 同比增速% |
| 贷款利率 | 车贷利率、金融方案成本 | 央行 | LPR利率 |
| 居民收入 | 可支配收入增长、购车预算 | 统计局 | 人均可支配收入增速 |
| 原材料价格 | 锂、钴、镍价格走势 | SMM/期货市场 | 元/吨 |
| 二手车市场 | 二手车市场成熟度、置换率 | 行业协会 | 置换率% |
| 汇率波动 | 人民币汇率、进口车成本 | 央行 | 汇率 |

### S - 社会文化 (Social)

| 分析维度 | 具体指标 | 数据来源 | 量化要求 |
|---------|---------|---------|---------|
| 城镇化率 | 城镇化率、出行方式变化 | 统计局 | 城镇化率% |
| 环保意识 | 环保意识、绿色购车偏好 | 调研报告 | 绿色购车偏好度% |
| Z世代 | Z世代购车偏好、品牌认知 | 调研报告 | 各品牌Z世代偏好度 |
| 家庭结构 | 家庭小型化、平均人口数 | 人口普查 | 平均家庭人口 |
| 共享出行 | 共享出行渗透率、网约车规模 | 交通部 | 渗透率% |
| 消费观念 | 提前消费意愿、储蓄率 | 统计局 | 消费者信心指数 |

### T - 技术 (Technology)

| 分析维度 | 具体指标 | 数据来源 | 量化要求 |
|---------|---------|---------|---------|
| 电动化路线 | 纯电/插混/增程销量占比 | 中汽协 | 各动力类型占比% |
| 智能驾驶 | L2/L3/L4渗透率、NOA开城数量 | 佐思汽研 | 渗透率% |
| 800V平台 | 800V高压平台车型数量 | 盖世汽车 | 车型数量 |
| 固态电池 | 固态电池商业化进展、量产时间表 | 行业新闻 | 量产时间 |
| OTA升级 | OTA功能迭代频率 | 媒体报道 | 升级频率 |
| 补能技术 | 充电桩密度、快充技术 | 充电联盟 | 充电桩数量/密度 |

## 3. 量化标准

| 评分 | 标准 | 说明 |
|------|------|------|
| 5分 | 极大利好/利空 | 政策强力支持/打压，市场显著受益/受损 |
| 4分 | 利好/利空 | 政策支持/限制，对市场有明显影响 |
| 3分 | 中性 | 政策影响有限，需要观察 |
| 2分 | 轻微利好/利空 | 影响较小 |
| 1分 | 可忽略 | 几乎无影响 |

## 5. 质量检查

**数据验证**：
- [ ] 所有数据都有明确来源
- [ ] 数据时间戳在6个月以内
- [ ] 政策文件已标注文号

**分析验证**：
- [ ] 各维度评分有具体数据支撑
- [ ] PEST与SWOT结论逻辑一致
- [ ] 评分考虑时间维度（短期/中期/长期）

## 6. 配合框架

**最佳组合**：
- PEST + SWOT：宏观环境 → 战略机会
- PEST + 波特五力：宏观环境 → 行业吸引力
- PEST + 4P：宏观环境 → 营销策略

**组合顺序**：
`
PEST分析 → 波特五力 → SWOT分析 → 战略建议
`
