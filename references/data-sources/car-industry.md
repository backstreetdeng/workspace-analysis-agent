# 汽车行业数据源配置

## 1. 数据源分类

### P0 - 核心数据源（必须获取）

| 用途 | 数据源 | 说明 |
|------|--------|------|
| 销量数据 | 乘联会月度/周度数据 | 最权威的零售销量数据 |
| 行业政策 | gov.cn/工信部/发改委 | 官方政策文件 |
| 财报数据 | 车企官方年报/季报 | 企业经营数据 |
| 行业新闻 | gasgoo.com（盖世汽车） | 行业动态 |

### P1 - 重要数据源（尽量获取）

| 用途 | 数据源 | 说明 |
|------|--------|------|
| 用户投诉 | 车质网 | 质量投诉数据 |
| 用户评价 | 汽车之家/懂车帝 | 用户口碑 |
| 保值率 | 精真估/车好多 | 二手车价格 |
| 电池数据 | 动力电池联盟 | 电池装机量 |
| 智驾数据 | 佐思汽研/高工智能汽车 | 智能驾驶渗透率 |

### P2 - 辅助数据源（按需获取）

| 用途 | 数据源 | 说明 |
|------|--------|------|
| 宏观数据 | 国家统计局 | GDP/人口/收入 |
| 金融数据 | eastmoney.com | 股价/估值 |
| 大宗商品 | SMM上海有色网 | 碳酸锂价格 |
| 海关数据 | 海关总署 | 进出口数据 |

## 2. 数据新鲜度要求

| 数据类型 | 最高新鲜度 | 最低可接受 | 过期处理 |
|---------|-----------|-----------|---------|
| 销量数据 | 月度 | 季度 | 使用估算 |
| 价格数据 | 实时 | 月度 | 标注时间 |
| 政策数据 | 实时 | 6个月内 | 需重新确认 |
| 财报数据 | 年度/季度 | 12个月 | 标注时间 |
| 用户口碑 | 月度 | 季度 | 参考趋势 |

## 3. 交叉验证规则

| 数据类型 | 验证要求 | 说明 |
|---------|---------|------|
| 销量数据 | 至少2个独立来源 | 乘联会 vs 中汽协 |
| 价格数据 | 官网 vs 终端 | 可能有差异 |
| 市场份额 | 多个数据源交叉 | 不同口径可能不同 |
| 政策数据 | 以官方公告为准 | 不采用非官方解读 |

## 4. 量化数据要求

| 数据类型 | 必须包含 | 示例 |
|---------|---------|------|
| 数值型 | 数值+单位+来源+时间 | ""销量: 150万辆(中汽协, 2024Q1)"" |
| 评分型 | 分数+置信区间 | ""竞争力: 4.2±0.3分"" |
| 趋势型 | 历史+预测+CAGR | ""CAGR: 15%(2021-2024)"" |
| 定性型 | 分析依据+置信度 | ""分析依据: 专家访谈, 置信度: 中"" |

## 5. 数据获取优先级

**市场规模分析**：
1. 中汽协/乘联会数据
2. 盖世汽车/36Kr
3. 车企财报

**竞品分析**：
1. 乘联会销量数据
2. 汽车之家/懂车帝报价
3. 财报/官网配置

**政策分析**：
1. gov.cn/工信部文件
2. 盖世/第一电动
3. 行业报告

**用户口碑**：
1. 车质网投诉
2. 汽车之家/懂车帝评价
3. 社交媒体
