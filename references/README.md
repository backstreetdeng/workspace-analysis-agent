## 使用指南

### AI Agent 使用流程

1. **读取相关框架指南**：根据问题类型读取对应框架文件
2. **执行分析**：按照框架要求收集数据和评分
3. **输出报告**：按照模板格式输出

### 框架选择逻辑

| 问题类型 | 推荐框架组合 |
|----------|-------------|
| 政策影响评估 | PEST + SWOT |
| 竞品分析 | 波特五力 + 竞品矩阵 + 4P |
| 市场机会识别 | TAM/SAM/SOM + SWOT |
| 综合分析 | PEST + 波特五力 + SWOT + 4P |

### 框架组合示例

**市场机会分析**：
`
1. PEST分析 → 宏观环境判断
2. TAM/SAM/SOM → 市场规模估算
3. SWOT → 机会与威胁识别
4. 竞品矩阵 → 差异化机会
`

**品牌竞争分析**：
`
1. 波特五力 → 行业竞争强度
2. 竞品矩阵 → 竞争格局定位
3. 4P分析 → 营销能力对比
4. SWOT → 战略建议
`
